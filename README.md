# PM.Journal

![Flutter](https://img.shields.io/badge/Flutter-3.x-blue?logo=flutter) ![Firebase](https://img.shields.io/badge/Firebase-Firestore-orange?logo=firebase) ![Language](https://img.shields.io/badge/Language-Dart-blue?logo=dart)

**PM.Journal** — это мобильное приложение для старост и преподавателей, предназначенное для удобного ведения журнала посещаемости учебной группы. Приложение построено на Flutter и использует Firebase для хранения данных в реальном времени.

## Основные возможности

*   **Отслеживание посещаемости**: Отмечайте студентов как присутствующих, отсутствующих или отсутствующих по уважительной причине.
*   **Динамическое расписание**: Автоматическое отображение пар для текущей учебной недели (четной/нечетной).
*   **Детализация пропусков**: Просматривайте полную историю отсутствий для каждого студента, включая дату, дисциплину и причину.
*   **Подпись отметок**: Каждая отметка о пропуске сохраняется с подписью пользователя, который ее сделал (например, ФИО старосты), что обеспечивает прозрачность.
*   **Управление студентами**: Простое добавление и просмотр списка студентов группы.
*   **Бэкенд на Firebase**: Все данные (студенты, расписание, отметки) хранятся в Firestore, что позволяет нескольким пользователям (например, старосте и его заместителю) работать с одними и теми же данными.
*   **Проверка обновлений**: Приложение при запуске проверяет наличие новой версии на GitHub Releases.

##  Технологический стек

*   **Фреймворк**: [Flutter](https://flutter.dev/)
*   **Язык**: [Dart](https://dart.dev/)

##  Порядок работы с приложением (Workflow)

1.  **Первый запуск**: Нажмите "Создать группу".
2.  **Настройка подписи (ВАЖНО!)**: Перейдите в `Настройки` -> `Подпись для отметок` и введите свое ФИО. Без этого сохранение отметок будет невозможно.
3.  **Добавление студентов**: Перейдите в `Настройки` -> `Студенты` и добавьте всех студентов вашей группы.
4.  **Ведение журнала**:
    *   Откройте вкладку `Отметки`.
    *   Выберите нужную дату и пару из выпадающего списка.
    *   Отметьте отсутствующих.
    *   Нажмите "Сохранить изменения". Ваша подпись будет автоматически прикреплена к каждой отметке.
5.  **Анализ пропусков**: Используйте вкладку `Пропуски` для просмотра сводной информации и детальной истории отсутствий по каждому студенту.
